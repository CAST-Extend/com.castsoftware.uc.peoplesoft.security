  -----------------------------------------
  CLI for the portable and lightweight Metrics Compiler.

  usage: MetricsCompiler -help
  Required Files:
	- MetricsCompilerCLI.jar
	- (optionally postgresql-9.2.1004.jdbc3.jar)
